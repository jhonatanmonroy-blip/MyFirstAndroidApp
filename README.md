# 📦 Proyecto UI Multilenguaje & Soporte 9-Patch

Bienvenido al proyecto 🚀  
En esta actualización se realizó la implementación de una nueva estructura basada en **9-Patch**, además de agregar soporte para múltiples idiomas con el objetivo de mejorar la escalabilidad, personalización y experiencia del usuario.

---

## ✨ Características Implementadas

### 🧩 Estructura 9-Patch
Se creó la estructura de **9-Patch** para permitir componentes visuales más flexibles y adaptables en diferentes resoluciones y tamaños de pantalla.

#### Beneficios:
- 📱 Mejor adaptación responsive
- 🎨 Escalado visual sin pérdida de calidad

---

## 🌍 Soporte Multilenguaje

El proyecto ahora cuenta con internacionalización (**i18n**) para los siguientes idiomas:

| Idioma | Código |
|--------|--------|
| 🇪🇸 Español | `es` |
| 🇺🇸 Inglés | `en` |
| 🇫🇷 Francés | `fr` |
| 🇩🇪 Alemán | `de` |

### ✅ Funcionalidades
- Cambio dinámico de idioma
- Traducciones centralizadas
- Escalabilidad para agregar nuevos idiomas fácilmente
- Mejor experiencia para usuarios internacionales

## 🚀 Objetivo de la Actualización

Esta implementación busca preparar el proyecto para:

- Obtener la certificacion de aplicaciones moviles de coursera

---

## 🛠️ Tecnologías Utilizadas

- Kotlin
- Android Studio
